==========================================
Welcome to PostgresML's documentation!
==========================================

``PostgresML`` is an end-to-end machine learning system. It enables you to train models and make online predictions
using only SQL, without your data ever leaving `your favorite database <http://postgres.org/>`_.


The code of this project lives on Github: `PostgresML <https://github.com/postgresml/postgresml>`_
